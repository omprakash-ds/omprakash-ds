## 📊 Projects

### 🚢 Titanic Survival Analysis
[View on GitHub](https://github.com/omprakash-ds/titanic-survival-eda)

End-to-end EDA on 891 passenger records identifying
key survival factors using Python and statistical
analysis.

**Key Insights:**
- Gender was strongest predictor: 74% female vs 19% male
- Class had direct impact: 1st class 63%, 3rd class 24%
- Family size 2-3 had highest survival rate
- Children under 10 showed notably higher survival

**Technical:**
- Missing value treatment across 3 columns
- Engineered FamilySize feature
- 13 visualizations across all key variables

`Python` `Pandas` `Seaborn` `Matplotlib`
`EDA` `Feature Engineering` `Statistics`
